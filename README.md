# markup-starter-kit
![twice](http://az879543.vo.msecnd.net/twice/20180701%200AM_TWICE1.jpg)

### Setting
* `npm install`
* `npm install gulp --save-dev`
* `npm install holderjs --save`

1. Create icon font - `gulp iconfont`
2. Create sprite - `gulp sprite`
3. start - `gulp`
### Info

  - Create icon font.
